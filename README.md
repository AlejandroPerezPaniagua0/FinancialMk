# FinancialMk

A robust financial application designed for **cross-market movement analysis**. FinancialMk enables the management of extensive historical databases and allows for precise comparisons between different dates and financial markets.

---

## Key Features

- **Multi-Market Analysis**: Tracking and comparison of real-time and historical movements.
- **Extensive Historical Data**: Capability to process and query large volumes of financial data.
- **Advanced Comparisons**: Integrated tools to contrast performance and flows between specific periods.
- **Professional Interface**: Clear visualization oriented towards decision-making.

## Project Structure

The project is organized to clearly separate business logic from the user interface:

- **`back/`**: Robust API developed in **Laravel 12**, handling business logic, data processing, and persistence.
- **`front/`**: Dynamic user interface for financial data exploitation and visualization.

## Tech Stack

### Backend
- **Laravel 12** (Core Framework)
- **PHP 8.2+**
- **SQLite / MySQL / PostgreSQL** (Data Persistence)

### Frontend
- **React**
- **TypeScript**
- **Vite** (Build tool)
- **Tailwind CSS** (UI/UX)

# FinancialMk

---

## Características Principales

- **Análisis Multi-Mercado**: Seguimiento y comparación de movimientos en tiempo real e históricos.

- **Comparativas Avanzadas**: Herramientas integradas para contrastar rendimientos y flujos entre periodos específicos.
- **Interfaz Profesional**: Visualización clara y orientada a la toma de decisiones.
