# FinancialMk - Backend API

This is the FinancialMk API, responsible for managing, storing, and querying historical financial market data to analyze performance.

---

## 吼 English

### Description
A robust API developed to collect historical data from financial markets. It enables precise queries regarding asset performance over time, serving as the data engine for the frontend.

### Features
- **Historical Data Management**: Efficient storage of market movements.
- **Performance Calculation**: Optimized logic for comparing periods and assets.
- **Clean Architecture**: Built with Laravel 12 to ensure scalability and security.

### Requirements
- PHP 8.2+
- Composer
- Database (PostgreSQL)

### Installation
1. Clone the repository.
2. Run `composer install`.
3. Configure the `.env` file (use `.env.example` as a reference).
4. Generate the application key: `php artisan key:generate`.
5. Run migrations: `php artisan migrate`.
